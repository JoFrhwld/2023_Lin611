Prepare the data for modelling

ey_dat |>

“Complete Pooling”

That is, pooling all speakers’ data together.

fit_lm <- function(df){
  lm(F1_n ~ freq_c, data = df)
}

Getting predictions

get_pred_tibble <- function(mod, freq_range){
  predictions(
    mod,
    newdata = datagrid(
      freq_c = seq(freq_range[1], freq_range[2], length = 10)

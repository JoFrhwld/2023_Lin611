R vs RStudio vs Quarto

R (on the left) is a programming language, which looks at (üëÄ) R scipt files (the box with .r) and interprets them. RStudio is an IDE (Integrated Development Environment) that provides a more pleasant interface to working with R.

Quarto (not pictured) is a program that interprets Quarto documents (the box with .qmd) and renders them as html files. We‚Äôll be doing this all inside of RStudio.
